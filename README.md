# El Pollo Loco

El Pollo Loco is a small browser game created as a learning project during my web development training.

The project was built with HTML, CSS, and JavaScript. The main goal was to practice object-oriented JavaScript, canvas rendering, keyboard and mobile controls, collision detection, sound handling, and responsive design.

## Features

- Canvas-based 2D jump-and-run gameplay
- Keyboard controls for desktop
- Mobile touch controls
- Collectable coins and salsa bottles
- Throwable bottle mechanic
- Endboss fight
- Background music and sound effects
- Mute button with saved local storage state
- Fullscreen support
- How-to-play dialog
- Responsive layout for smaller screens

## Controls

### Desktop

- `Arrow Left` / `Arrow Right`: Move
- `Space`: Jump
- `D`: Throw bottle
- `Enter`: Start game

### Mobile

Use the on-screen buttons to move, jump, and throw bottles.

## Technologies

- HTML
- CSS
- JavaScript
- Canvas API
- Local Storage
